# Maze Generator
Prosta aplikacja konsolowa w języku Java, która pozwala na generowanie, wyświetlanie oraz zapisywanie i wczytywanie labiryntów z pliku. Program wykorzystuje algorytm Prima do tworzenia losowych labiryntów.

### Plany na przyszłość
- Implementacja algorytmu wyszukiwania najkrótszej ścieżki (np. BFS lub Dijkstra).
